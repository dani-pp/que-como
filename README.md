# Recetario flask+mongodb

Aplicación web para guardar y consultar recetas. 

## Tecnologías

- Python + Flask
- MongoDB Atlas
- Tailwind CSS

## Instalación

1. Clona el repositorio
2. Copia '.env.example' como '.env' y configura tus variables
3. Ejecuta 
```bash
pip install -r requirements.txt
flask run
